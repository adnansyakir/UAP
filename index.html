Materi Pembelajaran

HTML
(HyperText Markup Language)

HTML adalah singkatan dari Hypertext Markup Language. HTML adalah bahasa markup standar untuk membuat dan menyusun halaman pada aplikasi website. HTML berfungsi untuk membuat struktur website hingga menyusun format teks dan gambar pada halaman web

CSS
(Cascading Style Sheets)

CSS adalah singkatan dari cascading style sheets. CSS adalah bahasa yang digunakan untuk mengubah dan memperindah tampilan dan format halaman website. pada CSS kita mengatur jenis font, warna tulisan, dan latar belakang halaman. CSS bertujuan untuk memisahkan konten dan style yang akan digunakan. 1 file CSS dapat digunakan untuk banyak halaman HTML
