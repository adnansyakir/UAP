OBJECT FIT

CSS object-fit adalah property css yang digunakan untuk <img> atau <video> supaya ukurannya bisa sesuai dengan box container. Semua borwser modern mendukung property ini.

Ada lima value yang bisa digunakan untuk object-fit, berikut tabel daftarnya:

  • contain - Ukuran konten disesuaikan dengan aspect rasio maksimal di box container
  • cover - Gambar akan dikrop dan dijadikan cover (memenuhi) box
  • fill - Ini default. Gambar “dipaksa” untuk sesuai dengan ukuran yang ditentukan
  • none - Konten tidak diubah ukurannya
  • scale-down - Ukuran seolah-olah tidak ditentukan (berefek kalau ukuran objek lebih kecil)
