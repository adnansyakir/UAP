Pendahuluan

CSS menyediakan beberapa properti yang memungkinkan kita untuk menentukan berbagai gaya teks seperti color, alignment, spacing, decoration, transformation dll.

Sifat-sifat teks yang umum digunakan adalah: properti text-align, text-decoration, text-transform, text-indent, line-height, letter-spacing, word-spacing dan banyak lagi. Properti ini memberi kita kontrol yang tepat atas tampilan visual karakter , kata , spasi , dan sebagainya.

Properti Text

Silahkan simak css property untuk formatting text.

  • Text color

    Dengan properti color kita bisa menggunakannya untuk memberikan warna pada teks. Nilai color dapat diatur oleh nama – tentukan nilai color nama, seperti red; Hex – tentukan nilai color hex, seperti #ff0000; RGB – tentukan nilai color RGB, seperti rgb (255,0,0); transparan; dll

    Contoh

  • Text Alignment

    Dengan properti text-align kita bisa menggunakannya untuk mengatur keselarasan teks secara horizontal. Kemungkinan nilai untuk properti ini adalah: left, right, center, justify dan inherit.

    Contoh

  • Text Decoration

    Dengan properti text-decoration kita bisa menggunakannya untuk mengatur hiasan teks yang bergaris. Kemungkinan nilai untuk properti ini adalah: none, underline, overline, line-through, blink dan inherit.

    Contoh

  • Text Transformation

    Text transformation digunakan untuk menentukan huruf besar dan huruf kecil dalam teks, atau memberikan huruf besar bagian pertama dari setiap kata. Untuk menggunakannya kita cukup memanggil properti: text-transform. Kemungkinan nilai untuk properti ini adalah: none, capitalize, uppercase, lowercase dan inherit.

    Contoh

  • Text Indentation

    Text-indentation digunakan untuk menentukan indentasi baris pertama atau lebih tepatnya memberikan lekukan pada teks pertama. Untuk menggunakannya kita cukup memanggil properti: text-indent.

    Contoh

  • Letter Spacing

    Letter spacing digunakan untuk menentukan ruang antara karakter dalam teks. Untuk menggunakannya kita cukup memanggil properti: letter-spacing. kita bisa menggunakan nilai negatif.

    Contoh

  • Line Height

    Line height digunakan untuk menentukan ruang antara garis. Untuk menggunakannya kita cukup memanggil properti: line-height.

    Contoh

  • Text Direction

    Text direction digunakan untuk mengubah arah teks. Untuk menggunakannya kita cukup memanggil properti: direction.

    Contoh

  • Word Spacing

    Word spacing digunakan untuk menentukan lebar dari sepasi dalam sebuah teks. Untuk menggunakannya kita cukup memanggil properti: word-spacing. Nilai negatif dibolehkan.

    Contoh

  • Text Shadow

    Text-shadow digunakan untuk menambahkan bayangan untuk teks. Cara menggunakannya kita cukup memanggil properti: text-shadow. Nilai negatif dibolehkan.

    Contoh

    Contoh diatas menentukan posisi bayangan: horizontal 4px, vertikal 2px, dan warna orange atau oren.
