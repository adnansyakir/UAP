Height Width

Pendahuluan

Pada CSS sendiri Kita bisa mengatur lebar dan tinggi pada elemen HTML dengan properti width dan height, width yang berarti lebar dan height yang berarti tinggi. Dengan begitu Kita bisa menyesuaikan ukuran lebar maupun tinggi sesuai dengan kebutuhan Kita. Secara default ukuran width dan height tergantung pada konten didalamnya.

Nilai width dan height

Pada properti width dan height Kita bisa memberikan nilai sebagai berikut:

  • auto– Ini adalah default. Browser menghitung width dan height secara otomatis
  • length – Mengatur width atau height dengan px, cm, pt dll
  • % – Mengatur width atau height dengan persen
  • initial – Mengatur width atau height ke nilai default
  • inherit – Mengatur width atau height ke nilai warisan dari elemen induknya

Width Height

Dengan properti width dan height Kita bisa memberikan ukuran pada suatu elemen. Width yang berarti lebar dan height yang berarti tinggi.

Max-width dan max-height

Dengan properti max-width dan max-height Kita bisa memberikan ukuran pada suatu elemen. Max-width yang berarti lebar maksimal dan max-height yang berarti tinggi maksimal.

Perhatikan contoh diatas !!

Box hijau diberi style yang membatasi maksimal lebar sebesar 200px dan maksimal tinggi yang diberi 50px. Jadi box yang berwarna hijau tidak akan bisa mempunyai lebar lebih dari 200px, tetapi bisa mempunyai lebar 200px atau dibawahnya, dan tinggi box hijau tidak akan bisa mempunyai tinggi lebih dari 50px, tetapi bisa mempunyai tinggi 50px atau dibawahnya. Berbeda dengan box berwarna kuning, yang tidak mempunyai batas lebar maksimal maupun tinggi maksimal.

Min-width dan min-height

Dengan properti min-width dan min-height Kita bisa memberikan ukuran pada suatu elemen. Min-width yang berarti lebar minimal dan min-height yang berarti tinggi minimal.

I C O N

Pendahuluan

Icon adalah gambar kecil di samping menu / teks (biasanya) untuk membantu visualisasi dari menu tersebut.

Di CSS umum digunakan icon yang sudah disediakan oleh vendor besar, yaitu:

  • Font Awesome
  • Bootstrap icon
  • Google icon

Font Awesome

Yang sering digunakan adalah font-awesome. Penggunaan dari icon ini juga sangat mudah. Ada banyak cara untuk menggunakan icon ini, kesempatan kali ini kita menggunakan jquery. Pasang script dibawah setelah kode <head> atau sebelum </head>.

<script src="https://use.fontawesome.com/releases/v5.13.0/js/all.js" data-auto-replace-svg="nest"></script>

Cara menggunakannya sangat mudah, yaitu tinggal menggunakan format:

<i class="fas fa-camera"></i>

fas adalah style icon tersebut, dan fa-camera adalah icon yang dipanggil.

Untuk list icon dapat kamu lihat di: https://fontawesome.com/icons

Bootstrap Icon

ika kamu sudah menggunakan template berbasis bootstrap maka kamu sudah bisa menggunakan ini. Tapi, kalau belum. Install dulu CSSnya, dengan cara panggil CSS eksternal dibawah ini. Taruh setelah kode <head>.

<link href="https://stackpath.bootstrapcdn.com/bootstrap/4.4.1/css/bootstrap.min.css" rel="stylesheet">

Cara menggunakannya sama seperti font awesome

<i class="glyphicon glyphicon-cloud"></i>

Ganti glyphicon-cloud dengan nama icon.List iconnya: https://icons.getbootstrap.com/#icons

T A B L E

Pendahuluan

Jika Kita membuat tabel pada HTML dan tidak diberikan style apa pun, maka browser menampilkannya tanpa border atau batas. Dengan begitu Kita perlu memberikan style atau CSS pada tabel agar memberikan design yang lebih menarik.

Menambahkan border pada tabel

Karena default dari tabel tidak ada visual perbatasan, maka Kita bisa memberikan visual perbatasan dengan properti border.

Tabel border collapse

Jika Kita telah melihat hasil dari contoh diatas, bahwa tabel memiliki border ganda dan ada sedikit ruang pada sekitar konten, ini karena border tabel dipisahkan secara default. Tetapi dengan properti border-collapse Kita bisa menghilangkanya supaya tabel memiliki border tunggal.

Table Layout

Secara default lebar dari tabel akan memperluas menyesuaikan lebar dari konten didalamnya, tetapi Kita bisa mengatur lebar tetap atau fixed pada tabel dengan properti table-layout. Kemungkinan nilai untuk properti ini adalah:

  • auto – dengan nilai auto lebar dari tabel akan memperluas menyesuaikan lebar dari konten didalamnya. Ini adalah default.
  • fixed – dengan nilai fixed lebar dari tabel akan mengikuti lebar yang telah ditentukan. Tata letak horizontal tabel tidak bergantung pada isi sel, tetapi hanya tergantung pada lebar tabel, lebar kolom, dan border atau jarak sel.
  • inherit – mengikuti nilai dari elemen induknya.

Table Empty Cells

Dengan properti empty-cells Kita bisa mengontrol border atau perbatasan apakah akan menggunakan border atau tidak pada sel tabel yang tidak memiliki konten didalamnya. Kemungkinan nilai untuk properti ini adalah:

  • show – menampilkan border atau perbatasan pada tabel yang tidak memiliki konten didalamnya. Ini adalah default
  • hide – tidak menampilkan border atau perbatasan pada tabel yang tidak memiliki konten didalamnya.
  • inherit – mengikuti nilai dari elemen induknya.

Tabel Border Spacing

Jika Kita menambahkan border pada tabel, maka tabel akan memiliki border atau perbatasan ganda, hal ini karena kedua elemen tabel <th> dan <td> memiliki border terpisah. Secara default border ganda tersebut mempunyai ruang didalam tabel. Kita bisa memperluas ruang tersebut dengan properti border-spacing.

Style Input

Pertama kita bisa mengedit input dulu. Pahami jenis-jenis input form supaya kita tahu yang diedit. Misal, jika input teks. Yaitu <input type="text"> maka selector cssnya adalah input[type=text].

Kode CSSnya adalah:

input[type=text] {width:100%;}

Jika ingin mengedit semua jenis inputnya, maka input[type=text]{} diganti dengan input {}. Selektor lain: input[type=password]{} , input[type=number]{}, input[type=submit]{} dll.

Atur Box Sizing

Box-sizing maksudnya atur layout box atau kotak dari input. Yang biasa diatur adalah:

  • width: mengatur panjang input
  • padding: mengatur jarak teks elemen di dalam input
  • margin: mengatur jarak antar elemen input
  • display: mengatur tipe penempatan tampilan
  • box-sizing: supaya border sudah termasuk ukuran box
